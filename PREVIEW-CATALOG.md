# Preview: catálogo horizontal editorial

**Rama:** `preview/catalog-premium`

## Diseño

Lista vertical de **cards horizontales** (foto izquierda · datos derecha), estilo mockup premium. Sin grid clásico ni toggle de vistas.

## Ver en local

```bash
git checkout preview/catalog-premium
npm run dev
```

Abre `/models`.

## Descartar

```bash
git checkout main
```
