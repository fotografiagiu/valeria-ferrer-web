## 🎯 **Parámetros Técnicos Recomendados**

### **Midjourney:**
- **Aspect Ratio:** `--ar 16:9` (para covers)
- **Quality:** `--q 2` (máxima calidad)
- **Style:** `--style raw` (más realista)
- **Version:** `--v 6.0` (última versión)
- **Stylize:** `--s 250` (balance realista-creativo)

### **DALL-E 3:**
- **Estilo:** Fotográfico realista
- **Calidad:** Alta (HD)
- **Relación:** 16:9
- **Iluminación:** Golden hour preferida

## ⚡ **Comandos Rápidos**

### **Midjourney:**
```bash
# Generación rápida
/imagine prompt: [PROMPT] --ar 16:9 --q 2 --style raw

# Variación
/vary [image_number]

# Remaster
/remix prompt: [PROMPT_MODIFICADO]
```

### **DALL-E 3:**
```python
# API call example
response = openai.images.generate(
    model="dall-e-3",
    prompt=PROMPT,
    size="1792x1024",
    quality="hd",
    style="natural"
)
```

## 📊 **Control de Calidad**

### **Checklist de Aprobación:**
- [ ] Resolución 1920x1080px mínimo
- [ ] Estilo consistente con marca
- [ ] Colores de marca presentes
- [ ] Composición equilibrada
- [ ] Iluminación profesional
- [ ] Sin elementos inapropiados
- [ ] Aspecto 16:9 correcto
- [ ] Calidad 4K ultra realista

### **Rechazo Automático:**
- Imágenes borrosas o pixeladas
- Colores fuera de paleta
- Elementos textuales no deseados
- Composición pobre
- Iluminación artificial
- Estilo cartoon/ilustrativo
